# Puissance4 / Connect 4  
  
A jquery plugin connect4  

## How to use  
  
All you need is a basic html file like this :  
  
`code(<!DOCTYPE html>
<html>
<head>
  <title>Connect4</title>
  <meta charset="UTF-8">
</head> 
<body>     
<script src="public/script/jquery-3.4.1.min.js"></script>
<script src="main.js"></script>
</body>
</html>)`
